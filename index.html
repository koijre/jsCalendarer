Calendarer

A basic calendar script. Easy to edit & customize for your project.

Parameters: showCalendar(boolean startWithMonday, array holidayDays, string returnFunction);

For holidayDays 0=Sunday, ..., 6=Saturday

Usage: Add a div with id parameter that equals calendar. Then call the calendar with the function showCalendar. You can change the name of days and months from script or writing on this page (like down below).On calendar onClick event for days will send string object.
